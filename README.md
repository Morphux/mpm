# Install
### Clone the repository
```
git clone https://github.com/Morphux/mpm.git
```

### Init
```
make init
```

### Compile
```
make
```

# Test
```
make check
```

# Documentation

### Dependencies
```
doxygen
```

Generate it:
```
make doc
```
